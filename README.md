## Usage

1. **Launch the Application**: Open the auto clicker tool you just downloaded.
2. **Set Click Interval**: Adjust the interval slider to set how fast you want the clicks to occur.
3. **Start Clicking**: Press the start button to begin the auto-clicking process.
4. **Stop Clicking**: Use the stop button to halt the clicking whenever you wish.

## Customization

You can tailor the auto clicker to your preferences:

- **Click Type**: Choose between left-click, right-click, or middle-click.
- **Hotkeys**: Set hotkeys to start and stop the clicking process easily.
- **Click Count**: Define how many clicks you want to execute before stopping automatically.
